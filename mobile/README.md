# Mobile app (Expo) — TimViecLam (mẫu)

This folder contains a minimal Expo React Native app skeleton for the TimViecLam demo.

How to run (locally):

1. Install Node.js and npm.
2. Install Expo CLI globally (optional):

   npm install -g expo-cli

3. From this folder, install dependencies and start:

   npm install
   npm run start

4. Use the Expo dev tools to open in the Expo Go app (phone) or run on an emulator.

Notes:
- This is a skeleton (no backend). Login is simulated in the client.
- To make a production native app, follow Expo or EAS Build docs, or eject to bare workflow and use native tooling.
